================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "Module not found: pandas"
Solution: pip install -r requirements.txt

Problem: "Failed to load data"
Solution: 
  1. Make sure you're running a web server
  2. Don't open index.html directly (file://)
  3. Use: python -m http.server 8000

Problem: "Excel file not found"
Solution: Ensure data/equities.xlsx exists

Problem: Changes not showing
Solution: Hard refresh browser (Ctrl+Shift+R)
